Semester Project 2

An interactive dashboard for tracking course milestones and deliverables.

Project Overview

Semester Project 2 consolidates course milestones into a single dashboard that helps you follow deadlines, track dependencies, and collaborate with teammates in real time.

  • Visualise milestones on a timeline to understand sequencing.
  • Assign owners and statuses so responsibilities stay clear.
  • Receive reminders before critical deliverables are due.

Core Features

  • Milestone timeline
  • Team assignments
  • Progress tracking
  • Automated reminders
  • Exportable reports

Getting Started

Begin by entering your course schedule, then invite teammates so everyone can contribute to the plan. Use the built-in templates to jump-start milestone planning and track submissions all semester long.
